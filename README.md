## 🎯 Sobre a Kivoo Studio

A **Kivoo Studio** é um studio de desenvolvimento de aplicativos mobile focado em resolver problemas reais de negócios. Não começamos pela tecnologia — começamos pelo problema.

Do levantamento de requisitos ao lançamento nas lojas, entregamos produtos digitais que funcionam de verdade.

---

## 🚀 O que fazemos

| Etapa | Descrição |
|-------|-----------|
| 🔍 **Briefing** | Entendemos fundo o problema do seu negócio antes de escrever uma linha de código |
| 🎨 **UI/UX Design** | Criamos o design de todas as telas e protótipo interativo no Figma |
| 📱 **Desenvolvimento** | Construímos o app para iOS e Android com tecnologia moderna e escalável |
| 🚀 **Lançamento** | Publicamos na App Store e Google Play e entregamos o produto funcionando |

---

## 🛠 Stack

```
Mobile          →  React Native / Flutter
Backend         →  Node.js / Firebase
Design          →  Figma
Versionamento   →  Git + GitHub
CI/CD           →  GitHub Actions
Deploy          →  App Store + Google Play
```

## 💡 Nossa filosofia

> *"Não existe app ruim. Existe problema mal entendido."*

Antes de qualquer linha de código, a gente:

1. **Ouve** — entende o problema real do negócio
2. **Mapeia** — levanta os fluxos do usuário
3. **Valida** — protótipo antes de desenvolver
4. **Constrói** — com qualidade e atenção a cada detalhe
5. **Entrega** — app nas lojas, funcionando
